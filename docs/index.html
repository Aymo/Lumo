Lumo

Lightweight Universal Moji OS — flash it to your board in 30 seconds.

⚠️ Your browser doesn't support Web Serial. Please use Chrome, Edge, or Opera on a desktop computer.

Install Lumo

Connect your Waveshare ESP32-S3-Touch-AMOLED-1.75 via USB and click below.

Version 0.1.0 · All releases

Before you start

  • You'll need a Waveshare ESP32-S3-Touch-AMOLED-1.75 board.
  • Use a USB-C cable that supports data (not just charging).
  • Use Chrome, Edge, or Opera on a desktop. Firefox and Safari don't support Web Serial.
  • Close any other program that might be using the serial port (Arduino IDE, PlatformIO monitor, etc.).

How it works

  1. Plug your board into your computer.
  2. Click the Install Lumo button above.
  3. Select your board's serial port from the popup.
  4. Click Install in the dialog and wait. The board will reboot when finished.

After flashing, your board will show Lumo's eyes. Tap the screen for emotions, swipe left for the menu.

Trouble?

  • No port shown? Try a different USB cable, or hold the BOOT button while connecting.
  • "Failed to connect"? Press and hold BOOT, click Install, release BOOT after 1 second.
  • Still stuck? Open an issue on GitHub.
